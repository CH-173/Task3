Сценарий: Почта успешно завершила работу
Дано: Программа просит сначала ввести <время> (в минутах), в течение которого работает почтовое отделение, затем просит ввести <количество> посетителей, ожидающих получения посылки, затем для каждого посетителя просит ввести <время> (в минутах), которое требуется для его обслуживания
Когда: Пользователь вводит целочисленное неотрицательное <время> в минутах, целочисленное неотрицательное <количество> посетителей и для каждого поочередно - целочисленное неотрицательное <время> обслуживания в минутах так, что общее время обслуживания всех посетителей (кроме последнего) не превышает время работы почты
То: Программа выводит сообщение "Все посетители получат посылки!"

Примеры:
|Время|Количество|Время|
|60   |2         |10   |
|     |          |15   |
------------------------
|40   |3         |15   |
|     |          |20   |
|     |          |35   |
