Сценарий: Вывод списка сохраненных форматов файлов
Дано: Программа выводит меню со списком <кнопок> и соответствующих им <разделах> меню
Когда: Пользователь выбирает первый пункт меню, нажимая соответствующую <кнопку>
То: Программа выводит список хранимых форматов файлов, состоящих из <короткого> наименования и <полной> расшифровки, затем предлагает пользователю вернуться в меню нажатием любой клавиши

Примеры:
|Кнопка|Раздел                          |
|1     |Вывести список форматов на экран|
|2     |Добавить новый формат           |
|3     |Удалить формат                  |
|4     |Науйти формат                   |
|5     |Выйти из программы              |

------------------------

|Короткое|Полное                   |
|.png    |portable network graphics|
|.exe    |executable file          |
|...     |...                      |

Сценарий: Выход из программы
Дано: Программа выводит меню со списком кнопок и соответствующих им разделах меню
Когда: Пользователь выбирает пятый пункт меню, нажимая соответствующую кнопку
То: Программа завершает свою работу


-----------------------------------------------------------------------

Сценарий: Выбран не существующий пункт меню
Дано: Программа выводит меню со списком кнопок и соответствующих им разделах меню
Когда: Пользователь нажимает любую кнопку, не соответствующую пункту меню
То: Программа не реагирует

Сценарий: Добавление нового формата в список
Дано: Программа выводит меню со списком кнопок и соответствующих им разделах меню
Когда: Пользователь выбирает второй пункт меню, нажимая соответствующую кнопку, затем вводит короткое и расширенное название нового формата (короткое название не совпадает с каким-либо из существующих элементов)
То: Программа добавляет введенные данные в список форматов и уведомляет пользователя об успешно выполненной операции, затем просит нажать любую клавишу для перехода в меню

Сценарий: Добавление существующего формата в список
Дано: Программа выводит меню со списком кнопок и соответствующих им разделах меню
Когда: Пользователь выбирает второй пункт меню, нажимая соответствующую кнопку, затем вводит короткое и расширенное название нового формата (короткое название совпадает с каким-либо из существующих элементов)
То: Программа соообщает о существовании в списке добавляемого элемента и предлагает выйти в меню

Сценарий: Добавление формата с пустым коротким или полынм наименованием в список
Дано: Программа выводит меню со списком кнопок и соответствующих им разделах меню
Когда: Пользователь выбирает второй пункт меню, нажимая соответствующую кнопку, затем вводит пустое короткое или пустое расширенное название нового формата
То: Программа сообщает о пустом значении и просит заново ввести короткое или расширенное название добавляемого формата

Сценарий: Удаление формата из списка
Дано: Программа выводит меню со списком кнопок и соответствующих им разделах меню
Когда: Пользователь выбирает третий пункт, затем вводит короткое наименование элемента (существующего в списке)
То: Программа удаляет формат из списка форматов и извещает пользователя об успешной операции, затем предлагает вернуться в меню

Сценарий: Поиск элемента по краткому наименованию

Когда: Пользователь выбирает четвертый пункт меню, затем вводит краткое наименование существующего в списке элемента
То: Программа выводит полное наименование искомого формата на экран и предлагает выйти в меню

Сценарий: Поиск не существующего элемента по краткому наименованию
